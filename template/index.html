Tài liệu Tổng quan

Mẫu tài liệu chuẩn hóa cho AI Agent

Chào mừng bạn đến với template tài liệu tối giản và hiện đại được tối ưu hóa cho cả người dùng lẫn các **AI Agent**. File HTML này chứa toàn bộ CSS và biểu tượng cần thiết qua CDN, sẵn sàng chạy ngay mà không cần cấu hình phức tạp.

Mục tiêu thiết kế

Tạo một khung chuẩn duy nhất giúp các AI agent tự động hiểu quy tắc UI, dễ dàng nhân bản mã nguồn để xây dựng hệ thống hướng dẫn, API Docs hoặc giáo trình lập trình đồng bộ.

Hướng dẫn dành cho AI Agent

Khi AI Agent muốn thêm nội dung mới, cần tuân thủ 2 vị trí neo (anchor comments) chính trong tệp tin:

  • 1
    Danh mục Sidebar: Thêm liên kết `` vào trong khối được bao bọc bởi `` và ``.
  • 2
    Nội dung chính: Thêm các thẻ `
    ` vào bên dưới thẻ bắt đầu ``.

Hộp thông báo (Alerts / Callouts)

Dùng để làm nổi bật các lưu ý, cảnh báo hoặc lời khuyên quan trọng.

Đây là một khối thông báo dạng Info được dựng sẵn bằng Web Component kế thừa Tailwind CSS.

Mã nguồn (Code Blocks)

Trình diễn mã nguồn lập trình với font chữ Courier/Mono và tích hợp sẵn chức năng Copy nhanh.

Bảng biểu & Dữ liệu (Tables)

Sử dụng để liệt kê tham số API, cấu hình, thông số kỹ thuật.

Tham số Kiểu dữ liệu Mô tả
configPath string Đường dẫn tương đối tới file config.json
autoCommit boolean Tự động đẩy mã nguồn sau khi tạo thành công.

Danh sách dạng Thẻ (Cards Grid)

Thích hợp làm danh mục bài viết nổi bật hoặc tính năng chính.

Tốc độ tối ưu

Sử dụng Tailwind CDN tải nhanh và dễ chạy ngay mà không cần cài đặt Node.js ban đầu.

Tính nhất quán

Mọi agent sẽ tuân thủ cấu trúc của template để duy trì trải nghiệm liền mạch.

Quy trình thực hiện (Steps Layout)

Trình diễn tuần tự các bước cài đặt hoặc hướng dẫn từng bước.

1

Khởi tạo Dự án

Chạy lệnh của NPM CLI để tự động kéo file template chuẩn về thư mục của bạn.

2

Tích hợp AI Agent

Cấu hình prompt của AI Agent để tham chiếu đến các anchor của file này.

Chuyển đổi tab (Tabs Switcher)

Dùng để chuyển đổi giữa các câu lệnh cài đặt hoặc các ngôn ngữ lập trình khác nhau.

npm install create-tailwind-docs
yarn add create-tailwind-docs
pnpm add create-tailwind-docs

Hộp mở rộng (Accordions / FAQs)

Thích hợp cho phần câu hỏi thường gặp (FAQs) hoặc nội dung ẩn/hiện nâng cao.

Hãy hạn chế tự viết các class CSS ad-hoc, thay vào đó hãy sử dụng lại hệ màu Slate/Indigo/Brand đã được cấu hình sẵn trong config Tailwind của tệp tin. Được. Bạn có thể thay đổi link Google Fonts tại phần Header và cập nhật cấu hình fontFamily trong đoạn script cài đặt của Tailwind CSS.
